# colgroup
> col 요소를 그룹으로 나누고 세로열 전체를 그룹화 할때 선언한다. 이 요소는 선택적으로 할 수 있지만 되도록 사용하는것을 권장한다.

**– HTML 경우**
``` html
<colgroup>
    <col width="100">
</colgroup>

<colgroup>
    <col span="2" width="100">
</colgroup>
```
**– HTML5 경우**
``` html
<colgroup>
    <col style="width:100px">
</colgroup>

<colgroup>
    <col span="2" style="width:100px">
</colgroup>
```
